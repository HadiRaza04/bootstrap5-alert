Bootstrap5-Alerts

Alerts

Bootstrap 5 provides an easy way to create predefined alert messages.

An Alert shows a message. Class: .alert
Success! Indicates a success message. Class: .alert alert-success
Info Indicates a info message. Class: .alert alert-info
Warning Indicates some warning. Class: .alert alert-warning
Danges Indicates something danger. Class: .alert alert-danger
Primary Indicates some primary message. Class: .alert alert-primary
Secondary Indicates some secondary message. Class: .alert alert-secondary
Light show some light. Class: .alert alert-light
Dark Indicates dark message. Class: .alert alert-dark

Alert Links

Add the alert-link class to any links inside the alert box to create "matching colored links".

Closing Alerts

The button with class="btn-close" and data-bs-dismiss="alert" is used to close the alert box.

The alert-dismissible class aligns the button to the right.
